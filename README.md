# Decline-Curve-Analysis
Decline Curve Analysis – Neelam–Heera Offshore Asset
	•	Conducted Decline Curve Analysis (DCA) for 9 wells using production data (oil, gas, and water) from the Neelam–Heera offshore asset to evaluate well performance.
	•	Applied exponential decline models in Excel to forecast future production and estimate well-wise and reservoir-level recovery factors.
	•	Determined the remaining productive life of each well by projecting production decline trends and estimating the economic production duration.
